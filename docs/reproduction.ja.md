## 一括再現

```bash
# Track B（ABM）アンカー：
cargo run --release -- reproduce --decision-mode rule_6dim --t-max 48 --seed 42

# Track A（心理測定）6 因子優位 + ΔR²（合成データ）：
uv run brinsfield-tools reproduce --sample synth --synthesize-n 300
```

`reproduce`（Rust）は定常動機 mix を reference と対比し PASS/off-anchor を表示する．`reproduce`（Python）は competing CFA を fit し，6 因子優位を確認，動機分布と Study 4 ΔR² を報告し `three_way_comparison.csv` を書き出す．
