# CLI reference

[English](cli.md) | [日本語](cli.ja.md)

The `brinsfield` binary has four subcommands.

## `run`

Single configuration. Writes `metrics.csv`, `motive_mix.csv`, `agents.csv`, `correlations.csv`, `llm_meta.json`, `config.json` to a timestamped results dir and refreshes `results/latest`.

| Flag | Default | Meaning |
|------|---------|---------|
| `--decision-mode` | `rule_6dim` | `llm` / `rule_6dim` / `rule_4dim` / `rule_3dim` |
| `--n-teams` / `--team-size` | 5 / 8 | organisation shape |
| `--n-levels` | 3 | hierarchical levels (descriptive) |
| `--network` | `watts-strogatz` | `watts-strogatz` / `erdos-renyi` / `barabasi-albert` |
| `--network-k` / `--network-beta` | 6 / 0.1 | WS degree `k` / rewire β (or ER `p`) |
| `--supervisor-homogeneity` | 0.0 | η_sup ∈ [0,1] toward a common supervisor openness |
| `--motive-init` | `0.35,0.20,0.13,0.13,0.13,0.06` | initial six-motive distribution |
| `--motive-learn-rate` | 0.10 | EMA η for `motive_dynamics` |
| `--psafety-learn` | 0.05 | ψ learning rate |
| `--p-retaliate` | 0.05 | per-agent per-step retaliation probability |
| `--shock-t` / `--shock-magnitude` | none / 0.3 | exogenous σ shock |
| `--temperature` / `--prompt-version` | 0.0 / 1 | LLM temperature / prompt template v1–v3 |
| `--t-max` / `--runs` / `--seed` | 48 / 1 / 42 | horizon / repeats / core seed |
| `--llm-seed` / `--llm-cache-path` | 0 / `.llm_cache/cache.json` | LLM seed offset / cache path |
| `--output-dir` | `results` | output base |

## `sweep`

Cartesian product over `--psafety-learn × --p-retaliate × --motive-init-defensive` (comma lists) × `--runs` seeds. Writes `sweep_summary.csv` + `sweep_config.json`.

## `ablate`

Run `--decision-modes rule_6dim,rule_4dim,rule_3dim` (comma list) side-by-side for `--runs` each; prints per-mode mean `KL(mix‖reference)` and defensive share, and writes `ablation_summary.csv`.

## `reproduce`

Runs one configuration and prints the steady-state motive_mix (mean over `t ≥ t_max/2`) against the Brinsfield reference, with PASS/off-anchor verdicts for the defensive 12.65% anchor, the ineffectual floor, and the deviant ceiling.
