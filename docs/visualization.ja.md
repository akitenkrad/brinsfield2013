# 可視化

[English](visualization.md) | [日本語](visualization.ja.md)

`brinsfield-tools` Python パッケージが ABM 出力を描画する．

## `visualize`（単一 run）

```bash
uv run brinsfield-tools visualize --results-dir results/latest
```

- `motive_mix_stack.png` — primary 動機比率の 6 領域積上げ面グラフ（時系列）．
- `silence_kl_timeseries.png` — step 別の沈黙率と `KL(mix‖reference)`．
- `motive_correlate_bar.png` — 最終 step の各動機と ψ（defensive/diffident/relational で負）・神経症（deviant/diffident で正）との Pearson r．

## `visualize-sweep`

```bash
uv run brinsfield-tools visualize-sweep --results-dir results/<ts>_sweep
```

- `sweep_defensive_heatmap.png` — ψ_learn × p_retaliate 上の defensive 比率．
- `sweep_kl_contour.png` — motive-init-defensive × p_retaliate 上の `KL(mix‖reference)` 等高線．
- `sweep_motive_response.png` — 初期 defensive 比率に対する定常動機比率（12.65% アンカー線付き）．

## `show-experiment-settings`

```bash
uv run brinsfield-tools show-experiment-settings --results-dir results/latest
```

run/sweep の設定テーブルと `llm_meta.json` ブロック（model / endpoint / 温度 / seed / cache-hit 率）を表示する．`--json` で機械可読出力．
